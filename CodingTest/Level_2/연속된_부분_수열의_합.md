## 연속된 부분 수열의 합

### Point
부분 수열을 직접 만드는 것이 아니라 인덱스와 부분수열의 합, 길이의 값으로 구할 수 있음.


---

```

class Solution {
    public int[] solution(int[] sequence, int k) {
        int[] answer = new int[2];
        int left = 0;
        int right = 0;
        int sum = 0;
        int len = Integer.MAX_VALUE;

        while (right < sequence.length) {
            sum += sequence[right];
            right++;

            while (sum > k) {
                sum -= sequence[left];
                left++;
            }

            if (sum == k && right - left < len) {
                len = right - left;
                answer[0] = left;
                answer[1] = right - 1;
            }
        }

        return answer;
    }
}
```
